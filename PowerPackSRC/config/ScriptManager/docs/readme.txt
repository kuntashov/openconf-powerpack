	"Менеджер скриптов" (ScriptManager) - это набор скриптов и скриптлетов
для ОпенКонфа, расширяющие его сервисный функционал  в части управления
скриптами и назначенными клавиатурными сочетаниями (хоткеями) для макросов,
которые реализуются этими скриптами.

Основные функциональные возможности

	* Загрузка скриптов из заданных директорий
		-- список директорий задается в файле Folders.lst, который должен
			быть расположен в директории ScriptManager; см. п. "Загрузка скриптов";
		-- поддерживаются фильтры исключений для загружаемых скриптов:
			фильтр - это регулярное выражение, применяемое к полному
			пути файла; если полное имя файла удовлетворяет данному фильтру,
			то скрипт не загружается; см. файл install.txt;

	* Поддержка контекстно-зависимых хоткеев
		-- возможность назначения на одну и туже комбинацию клавищ (хоткей)
			нескольких макросов, которые будут исполняться в зависимости
			от контекста, в котором он будет вызываться (контекст определяется
			текущим активным окном, например активное окно редактора текстов
			образует один контекст, активное окно редактора диалогов - другое,
			активное окно конфигурации - третье и т.п. (см. файл hkedit.txt)
		-- редактирование клавиатурных сокращений в диалоговом режиме (см.
			файл hkedit.txt)

	Подробно принципы функционирования Менеджера скриптов и способы его
настройки описаны в следующих файлах (располагаются в каталоге
документации Менеджера скриптов, ScriptManager\docs):

	* design.txt - описание принципов работы Менеджера скриптов
	* install.txt - инструкция по установке и быстрой настройке
	* hkedit.txt - инструкция по использованию "Редактора хоткеев"
