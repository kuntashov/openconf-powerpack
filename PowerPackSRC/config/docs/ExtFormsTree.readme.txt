ExtFormsTree
---
Плагин для конфигуратора, реализующий отображение содержимого папок в виде дерева в окне конфигурации.

список обрабатываемых каталогов хранится в каталоге плагинов под именем: ExtFormsTree.txt

Основные возможности:
- отображение в виде дерева содержимого папок, может использоваться маска при заполнении
(*.ert,*.efd) разделителем масок является запятая (",")
- двойном щелчке на дереве открывается выбранный файл, если файл из разряда 1С (*.ert,*.mxl,*.txt) открывается в окне 1С, 
в других случаях связанные с данным расширением приложения.
- создание новых каталогов в текущем "Дереве"
- переименование,удаление файлов, каталогов
- по Drag N Drop можно копировать, перемещать файлы, каталоги

Плагин реализует лишь то, что нужно было мне, 
поэтому не даю гарантии, что в дальнейшем буду развивать его.
но если будут хорошие идеи, которые не потребуют большого количества времени - возможно будут реализованы.
