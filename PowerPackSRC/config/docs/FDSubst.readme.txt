Плагин для OpenConf 
"Замена стандартных диалогов открытия/сохранения файла"

Автор: metaeditor <shotfire@inbox.ru>

ОПИСАНИЕ:
  - Плагин "превращает" все стандартные диалоги открытия/сохранения файлов 
    в Explorer-Style диалоги (с Places bar панелью слева).
  - В левой панели имеется возможность настроить две дополнительных кнопки 
    помимо пяти стандартных. 
    (которые также настраиваются - http://www.script-coding.info/CommonDialogBox.html).
  - Запоминаются пути последних открытых/сохранённых файлов.
  - Внизу диалога добавлен статус бар, отображающий текущую открытую папку.
  - В список фильтров диалога открытия файла добавлен пункт "Все файлы"
  - В верхней панели диалога добавлена настраиваемая кнопка "Избранное"  
  - При удержании клавиши Shift в момент открытия диалога, открывается диалог прежнего вида.

УСТАНОВКА:
  Скопировать файл fdsubst.dll в <Каталог установки 1С>\BIN\Config

НАСТРОЙКА ДОПОЛНИТЕЛЬНЫХ КНОПОК:
  Для этого необходимо в ветви реестра
  HKEY_CURRENT_USER\Software\1C\1Cv7\7.7\OpenConf\Plugins\FDsubst\CustomPlaces\
  создать два строковых параметра с именами CustomPlace1 и CustomPlace2
  со значением формата <Заголовок кнопки>#<Путь к папке> 
  например Projects#D:\1C\Projects
  Если значения в реестре не заданы, то по умолчанию создаются кнопки:
  "Каталог ИБ" и "Каталог 1С"
  В пути допускаются символы макроподстановки и переменные окружения, подробнее см. ниже. 

НАСТРОЙКА ИЗБРАННОГО:
  Для этого необходимо в ветви реестра
  HKEY_CURRENT_USER\Software\1C\1Cv7\7.7\OpenConf\Plugins\FDsubst\MRUList\
  задать список строковых параметров, где:
    Имя параметра - это наименование пункта выпадающего меню
    Значение параметра - сопоставляемый ему путь к папке или файлу.
    В пути допускаются символы макроподстановки и переменные окружения, подробнее см. ниже. 
    
НАСТРОЙКА ПУТИ К ПАПКАМ:
  В строке пути могут быть указаны значения переменных окружения, т.е %windir%, %userprofile%, %programfiles%, и т.д...
  Также в качестве "переменных окружения" можно указывать %ibdir% и %bindir% - каталог информационной базы и 
  каталог программных файлов 1С соответственно.
  Кроме того, перед значениями переменных окружения допускаются символы "-", количество которых
  обозначает переход на это количество уровней назад.
  Примеры строк и возможных значений:
    %-userprofile%\All Users\Documents - общие документы
    %ibdir%\ExtForms - каталог внешних обработок текущей базы
    %-ibdir% - каталог с базами
    %bindir%\Config\Intell - папка с файлами интелла
    %bindir%\Config - папка опенконфа
    %bindir%\Config\Scripts - папка со скриптами
    %temp% - каталог временных файлов
    %-windir% - системный диск

Р.S: при щелчке мышью по кнопке "Избранное" удерживая Shift, происходит обновление настроек из реестра.
