Проект Visual 1C++
Автор: Александр Орефков orefkov@ezmail.ru
Проект представляет из себя плагин к Конфигуратору.
Назначение:
Оболочка для облегчения работы с классами 1С++.
Для работы плагина необходимо использовать загрузчик
плагинов "Открытый Конфигуратор".
Использование:
разместите vis1cpp.dll в папке "bin\config"

Плагин написан в тесном сотрудничестве с Дмитрием
Гусевым aka deb, автором 1С++, за что ему огромное
спасибо. Без его помощи плагин увидел бы свет гораздо
позже.

Текщие возможности оболочки:
Отображение в дополнительной закладке окна Конфигурации
дерева классов 1С++, с указанием методов и переменных,
а также веткой базовых классов.
Классы группируются по файлам объявлений.
По щелчку или Enter'у на методе класса происходит переход
к реализации метода.
Работают контекстные меню, копирование названий в буфер обмена.
Методы, прописанные в заголовке файла, отображаются в дереве
всегда, независимо от наличия реализации,
остальные методы и переменные динамически вытаскиваются
из модуля класса.
Динамически - это значит, что при открытии модуля в Конфигураторе,
текст модуля считывается из рабочего документа, а не из файла.
Динамический разбор defcls.prm пока не реализован, то есть
при изменении в файлах объявлений необходимо вручную обновить
дерево классов.
