MetaDataObjProps::Name параметр Rus сделан опциональным, по умолчанию True.
MetaDataObjChilds::Name - параметр Rus сделан опциональным, по умолчанию True.
MetaDataObj::DescribeObject параметр WithChilds сделан опциональным, по умолчанию False.
Message - маркер сделан опциональным, по умолчанию mNone.
Quit - параметр PromtSave сделан опциональным, по умолчанию True.
MetaData::SaveMDToFile, TestRefers - сделаны опциональные параметры.
LogMessageWrite - параметры ТипСобытия и Событие сделаны опциональными.
Run1CApp - вариант запуска 1С сделан опциональным, по умолчанию "Предприятие".

Document.Name и Documents.item(Name) приведены в соответсвие друг другу.
Для таблиц форм Document.Name изменено с ".Таблица.Номер" на ".Имя Таблицы"
Для общих таблиц Document.Name изменено с "Таблицы.НомерТаблицы" на "Общие Таблицы.Имя Таблицы"
Для ГМ Name="Глобальный модуль", однако Documents("ГлобальныйМодуль") для совместимости тоже работает.
Document.Name для описаний возвращается в виде "Описание." ID объекта "." полное имя объекта
Для получения описания через Documents.Name достаточно указать "Описание." ID объекта
Documents.item(Name) исправлено получение форм списка плана счетов по имени.

Исправлена ошибка чтения периодичности константы.
