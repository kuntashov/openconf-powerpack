Версия библиотеки dynwrap.dll для операционных систем семейства Win 95/98/ME.

Данная DLL реализует COM-компонент DynaWrap, позволяющий вызывать функции
Win32 API из скриптов, подерживающих технологию Microsoft Active Scripting.

Подробности об установке и регистрации см. в файле ../../docs/System.txt,
раздел посвященный dynwrap.dll.
