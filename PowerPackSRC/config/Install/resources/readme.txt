Все компоненты устанавливаемого программного продукта, 
включая основные файлы ОпенКонф, скрипты, скриптлеты и 
dll-файлы поставляеются по принципу "As Is" ("Как есть").

Авторы ни одного из включенных в поставку компонентов 
не несут никакой ответственности в случае потери информации 
в результате использования какого либо из компонентов.

Авторские права на OpenConf и плагин Телепат 2.0r 
принадлежат Александру Орефкову.

Авторские права на скрипты и компоненты, включенные в 
поставку принадлежат OpenConf Community, если это не оговорено
специально.

Авторские права на COM-компоненты, вошедшие в поставку перечислены
в файле System.readme.txt (config\docs\System.readme.txt).
